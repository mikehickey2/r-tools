`if,else`: testing a condition 
`for`: execute a loop for a fixxed number of times
`while`:execute a loop while a condition is true
`repeat`:execute an infinate loop
`break`: break the execution of a loop
`next`: skip an iteration of a loop
`return`: exit a function

Typically used when writing functions or longer expressions

** `if, else` control structure **

```
if(<condition>) {
	# do something
} else {
	# do something else
} 
```
example:
```
if(x > 3) {
	y <- 10
} else {
	y <- 0
}
```

** `For` loops **

`for` loops take an iterator variable and its successive values from a sequence or vector.

```
for(i in 1:10) {
	print(i)
}
```
This example gives the variable i a number and prints it out 1 through 10
